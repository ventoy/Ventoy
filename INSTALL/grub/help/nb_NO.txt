 L - Velg språk
F1 - Vis hjelpeinformasjon
F2 - Bla gjennom og start filer på den lokale disken
F3 - Bytt menymodus mellom Trevisning <-> Listevisning
F4 - Start Windows/Linux på den lokale disken
F5 - Verktøy
F6 - Last egendefinert Grub2-meny
F7 - Bytt mellom GUI-modus <-> TEKST-modus

m/Ctrl+m - Sjekksummér bildefiler (md5/sha1/sha256/sha512)
d/Ctrl+d - Memdisk-modus (Kun for små WinPE/LiveCD ISO/IMG)
w/Ctrl+w - WIMBOOT-modus (Kun for Windows/WinPE ISO-filer)
r/Ctrl+r - Grub2-modus (Kun for noen Linux-distribusjoner)
i/Ctrl+i - Kompatibilitetsmodus (Kun for feilsøking)
u/Ctrl+u - Last ISO EFI-driver (Kun for feilsøking, kan ikke brukes offisielt)
